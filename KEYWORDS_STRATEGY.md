## üí° Keyword Research Tools to Use

### Free Tools
1. **Google Search Suggestions**
   - Type keyword in Google
   - See suggestions dropdown
   - Use for long-tail ideas

2. **Google Search Console**
   - Monitor actual search queries users use
   - See impressions and clicks
   - Find keywords you rank for already

3. **Google Trends**
   - See keyword popularity over time
   - Seasonal trends
   - Related queries

4. **AnswerThePublic**
   - Free tier shows questions people ask
   - Find question-based keywords
   - Build FAQ section

5. **Ubersuggest Free**
   - 3 free searches/day
   - Keyword volume estimates
   - Related keywords

### Paid Tools (Optional)
- **Ahrefs**: Best overall keyword research
- **SEMrush**: Comprehensive keyword analysis
- **Moz Keyword Explorer**: User-friendly
